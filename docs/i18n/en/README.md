# SomaBrain Documentation (English)

This directory contains the English version of the SomaBrain documentation.

## Structure

- `user-manual/` - End-user documentation
- `technical-manual/` - System architecture and operations
- `development-manual/` - Developer guides
- `onboarding-manual/` - New team member onboarding
- `agent-onboarding/` - AI agent integration guides

## Language Support

Currently supported languages:
- **English** (en) - Primary language

Planned languages:
- German (de)
- Spanish (es)
- French (fr)

## Translation Guidelines

When translating documentation:

1. **Preserve Code Examples**: Keep all code blocks in English
2. **Maintain Structure**: Keep the same file/folder structure
3. **Update Links**: Ensure internal links point to translated versions
4. **Technical Terms**: Use glossary for consistent terminology
5. **Version Sync**: Keep translations in sync with English version

## Contributing Translations

1. Create language directory: `docs/i18n/{lang_code}/`
2. Copy English structure
3. Translate markdown content
4. Update `front_matter.yaml` with language selector
5. Submit pull request
