# Documentación de SomaBrain (Español)

Este directorio contiene la versión en español de la documentación de SomaBrain.

## Estructura

- `user-manual/` - Documentación para usuarios finales
- `technical-manual/` - Arquitectura del sistema y operaciones
- `development-manual/` - Guías para desarrolladores
- `onboarding-manual/` - Incorporación de nuevos miembros del equipo
- `agent-onboarding/` - Guías de integración para agentes IA

## Soporte de Idiomas

Idiomas actualmente soportados:
- **Inglés** (en) - Idioma principal
- **Español** (es) - Traducción completa

Idiomas planificados:
- Alemán (de)
- Francés (fr)

## Directrices de Traducción

Al traducir documentación:

1. **Preservar Ejemplos de Código**: Mantener todos los bloques de código en inglés
2. **Mantener Estructura**: Conservar la misma estructura de archivos/carpetas
3. **Actualizar Enlaces**: Asegurar que los enlaces internos apunten a versiones traducidas
4. **Términos Técnicos**: Usar glosario para terminología consistente
5. **Sincronización de Versiones**: Mantener traducciones sincronizadas con versión en inglés

## Contribuir Traducciones

1. Crear directorio de idioma: `docs/i18n/{código_idioma}/`
2. Copiar estructura en inglés
3. Traducir contenido markdown
4. Actualizar `front_matter.yaml` con selector de idioma
5. Enviar pull request
