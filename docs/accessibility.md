# Accessibility Statement (WCAG 2.1 AA Compliance)

## Commitment

SomaBrain documentation is committed to WCAG 2.1 Level AA accessibility standards.

## Compliance Status

**Current Status**: Partial Compliance

### Perceivable

- ✅ All code examples include syntax highlighting
- ✅ Configuration examples use clear, readable formatting
- ✅ Diagrams will include alt text (when added)
- ⚠️ Color contrast ratios to be verified in rendered documentation

### Operable

- ✅ All documentation is keyboard-navigable
- ✅ No time-based interactions required
- ✅ Clear heading hierarchy (H1 → H2 → H3)
- ✅ Skip-to-content links available in MkDocs theme

### Understandable

- ✅ Plain language used throughout
- ✅ Technical terms defined in glossary
- ✅ Consistent navigation structure
- ✅ Error messages include recovery steps

### Robust

- ✅ Valid Markdown syntax
- ✅ MkDocs generates valid HTML5
- ✅ Compatible with screen readers
- ✅ Works across modern browsers
