Security Policy

Thank you for reporting security issues in SomaBrain.

Reporting
---------
If you discover a security vulnerability, please responsibly disclose it by emailing security@somatechlat.com (or the project maintainer listed in CODEOWNERS) with details, reproduction steps, and a suggested mitigation. Do not open a public issue for security problems.

Response
--------
We will acknowledge receipt within 3 business days and provide an estimated timeline for a fix. For severe vulnerabilities we will coordinate privately and issue a CVE if appropriate.

Disclosure
----------
We follow coordinated disclosure practices. After a fix is available and users have had reasonable time to upgrade, we will publish an advisory describing the issue and mitigation steps.
